# Pinboard Pro

> Google Chrome extension for Pinboard.in (https://pinboard.in).

![Pinboard Pro](preview.jpg)

## Install

Install from Chrome Web Store.

## Development

Install dependencies:

```bash
$ npm install
```

Local debugging:

```bash
$ grunt debug
```

Lint JavaScript (using ESLint):

```bash
$ grunt lint
```

## Deployment

Build extension:

```bash
$ grunt build
```
